# Titanic Dashboard

Created an interactive Power BI dashboard using the Titanic dataset to analyze passenger and survival patterns.

## Files
- clean_titanic.csv (dataset)
- Titanic_Dashboard.pbit (Power BI dashboard template)

## Work Done
- Imported dataset into Power BI  
- Created interactive charts and visualizations  
- Added slicers and filters for dynamic analysis  
- Analyzed survival rate, passenger class, gender distribution, age, and fare patterns  

## Output
- Interactive dashboard with meaningful visual insights and filters

## Tools Used
- Power BI  
- CSV Dataset
